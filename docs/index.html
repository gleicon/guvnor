Process Manager & Reverse Proxy in One Binary

Auto-detects your apps, manages processes, routes requests, and handles HTTPS. Zero configuration required.

~/my-project
$ cd my-project
$ guvnor init
✓ Detected Node.js app
✓ Generated guvnor.yaml
$ guvnor start
✓ Started my-project on http://my-project.localhost:8080
curl -sSL https://github.com/gleicon/guvnor/releases/latest/download/guvnor-$(uname -s)-$(uname -m) -o guvnor

What Guvnor Does

Auto-Detection

Scans Node.js, Python, Go, Rust, PHP, Java projects

Process Management

Starts, stops, restarts, monitors health automatically

Virtual Hosts

Routes requests by hostname to correct app

Automatic HTTPS

Let's Encrypt certificates with zero configuration

Single Binary

No Docker, nginx, or complex setup needed

Heroku Compatible

Works with existing Procfiles and .env files

Quick Examples

New Project

mkdir my-app && cd my-app
echo 'console.log("Hello")' > server.js
guvnor init && guvnor start

Visit http://my-app.localhost:8080

Existing Project

cd my-project
guvnor init
guvnor start

Auto-detects your app and generates config

Multiple Apps

apps:
  - name: web
    command: npm
  - name: api
    command: python
    args: ["api.py"]

Each app gets its own hostname

Production

guvnor start --domain myapp.com \
  --email admin@myapp.com

Automatic HTTPS with Let's Encrypt
